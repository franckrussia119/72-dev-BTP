# 72DEV-BTP SARL Website

Next.js website for 72DEV-BTP SARL construction company.

## Deploy on Coolify

1. Push this repo to GitHub
2. In Coolify: New Resource > Public Git Repository
3. Paste GitHub URL
4. Build Pack: Nixpacks or Dockerfile
5. Port: 3000
6. Deploy

## Local Development

```bash
npm install
npm run dev
```
